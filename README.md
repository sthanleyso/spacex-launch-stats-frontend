## Overview

Codebase for [https://spacex-launch-stats.netlify.com/](https://spacex-launch-stats.netlify.com/) an app that lists SpaceX missions stats.
Current iteration is built using ReactJS, GraphQL and Netlify.

### Features

- Netlify integration
- GraphQL
- Apollo Client
- Styled components
